# NYC Taxi Trips - Data Analysis in R

This repository contains an R case study project analysing New York City taxi trip data. The project implements a full pipeline workflow for data ingestion, cleaning, preparation, and reporting.

View the final report: <a href="https://anw-g01.github.io/nyc-taxi-trips/" target="_blank">NYC Taxi Trips Report</a>

## Repository Structure

### Execution Scripts
- `main.R`: Main script to run the full data workflow sequentially.

### Project Directories
- `data/`: Stores raw and cleaned datasets used in the project.
- `scripts/`: Contains R scripts for data importing, cleaning, and preparation.
- `reports/`: Contains Quarto markdown files for exploratory data analysis and final reporting.
- `outputs/`: Generated summary tables (CSV) and figure images from the analysis.
- `docs/`: HTML output and supporting files generated by Quarto for the main report.

## Run the Project

Clone the repository:

```bash
git clone https://github.com/anw-g01/nyc-taxi-trips.git
cd nyc-taxi-trips
```

Install required R packages and restore the project environment using renv: `renv`:
```r
renv::restore()
```

Execute the full data workflow by running:
```r
source("main.R")
```

Render and view Quarto documents under the `reports/` directory.
