Author

Anwarat Gurung

Published

Thu, 12 Mar 2026

This report presents a series of visualisations and analyses based on the cleaned NYC Taxi Trip (2021) dataset, available here. The outputs focus on trip activity and patterns, including taxi vendor performance, revenue trends, tipping behaviour, and hourly peak travel periods.

1 Revenue by Taxi Vendor

Which taxi vendor accumulated the most revenue in 2021?

While revenue per mile is slightly higher for Green Taxi, Uber recorded significantly greater trip volume and total revenue in 2021, along with higher revenue per trip.

The full table of metrics is shown below the figure.

# aggreagate and summarise data
metrics_by_vendor <- TAXI_TRIPS_NYC %>% 
    group_by(vendorid) %>% 
    summarise(
        # aggregate key metrics

1.1 Daily Revenue Trends

How has revenue changed over 2021 for both vendors?

Revenue for both vendors remained broadly stable throughout 2021, with short-term fluctuations, while Uber showed a slight upward trend.

# summarise daily revenues for Green Taxi and Uber only
daily_revenues <- TAXI_TRIPS_NYC %>% 
    filter(vendorid %in% c("Green Taxi", "Uber")) %>% 
    group_by(vendorid, pickup_date) %>%
    summarise(
        daily_rev = sum(total_amount), 
        .groups = "drop"
    )     

ggplot(
    data = daily_revenues,
    mapping = aes(
        x = pickup_date,
        y = daily_rev,
        colour = vendorid       # group by vendorid for separate lines
    )
) +
    # line plot (series)
    geom_line(
        # alpha = 0.3,
        linewidth = 0.6
    ) +
    
    # prevent clipping of any labels
    coord_cartesian(clip = "off") +        

# hardcoded date filtering
    scale_x_date(
        limits = as.Date(c("2021-07-01", "2021-07-31"))
    ) +
    
    geom_smooth(
        method = "lm",
        se = FALSE,
        linetype = "dashed",
        alpha = 0.4,
        linewidth = 0.6
    ) + 
    
    scale_y_continuous(
        n.breaks = 10,
        labels = scales::label_number(
            prefix = "$",
            scale = 1e-3,
            suffix = "k",
            accuracy = 1
        )

    # manually set colour for each taxi vendor
    scale_colour_manual(
        values = c(
            "Green Taxi" = COLOUR_GREEN,
            "Uber" = COLOUR_UBER

    ) +

    # set text for: title, subtitle, caption

        title = str_c(

2 Average Total Green Taxi Spend by Weekday

What is the average amount spent on Green Taxi trips across the week, and are any weekdays significantly higher than others?

On average, total amounts were slightly higher on Sunday and Thursday, with the lowest on Saturday, but differences across the week were very minimal.

  • Data
